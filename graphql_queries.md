#### Get single client
``` 
{
  client(id:"1"){
    id
    name
    email
    phone
  }
}
```

#### Get All clients
``` 
{
  clients{
    id
    name
    email
    phone
  }
}
```

#### Get single project
``` 
{
  client(id:"1"){
    id
    name
    email
    phone
  }
}
```

#### Get All projects
``` 
{
  projects {
    name
    status
    description
  }
}
```
